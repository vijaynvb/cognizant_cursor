# Example 3 — Hotel Accommodations Page

A responsive hotel accommodations page built with **HTML**, **CSS**, **JavaScript**, and **Bootstrap 5**.

## Features

- Three room cards matching the reference design (De Luxe Room, De Luxe Sea View, Wellhall Family Suite)
- Hover effects with image zoom and "View Details" overlay
- Bootstrap modal with room description and "Book Now" button
- Fully responsive layout (single column on mobile, three columns on desktop)
- Warm, luxury color palette with serif + sans-serif typography

## How to Run

Open `index.html` directly in a browser — no build step required.

```
# or serve with any static file server
npx serve .
```

## Tech Stack

| Layer   | Tool              |
|---------|-------------------|
| Layout  | Bootstrap 5.3     |
| Fonts   | Google Fonts (Playfair Display, Lato) |
| JS      | Vanilla ES6       |
| Images  | Unsplash (hotlinked) |
